## Overview

**DGV-Labeler** is a lightweight multi-task assisted auto-labeling framework designed for dairy goat surveillance videos in real farming environments. The framework aims to reduce the high labor cost of manual video annotation by generating multiple types of labels from raw videos, including **bounding boxes**, **instance masks**, **track IDs**, and **behavior categories**.

This repository is associated with the manuscript:

> **DGV-Labeler: A Lightweight Multi-task Auto-labeling Framework for Dairy Goat Video Annotation**

The full implementation code will be released after the paper is accepted.
